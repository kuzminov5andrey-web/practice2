Инструмент для визуализации графа зависимостей пакетов Ubuntu.

# Этап 1: Конфигурация
- Чтение настроек из XML файла
- Валидация параметров конфигурации
- Вывод параметров в формате ключ-значение

# Этап 2: Сбор данных  
- Получение информации о зависимостях из APT репозиториев Ubuntu
- Поддержка указания версии пакета
- Извлечение прямых зависимостей по заданному URL репозитория

# Этап 5: Визуализация
- Генерация PlantUML диаграмм зависимостей
- Визуализация для трех пакетов (nginx, python3, git)
- Показ общих зависимостей между пакетами

# Использование

1. Настройте `config.xml`
2. Запустите: `python deps.py`
3. Скопируйте `graph.puml` на plantuml.com для получения PNG
